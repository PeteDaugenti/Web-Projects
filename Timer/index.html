JavaScript FAQs

What is JQuery?

JQuery is the library of some of the most used JavaScript functions you'll most likely use in a professional environment.

Why is JQuery ecomming so popular?

Three reasons:

  • It's free.
  • It lets you get more done in less time.
  • All of its functions are cross-browswer compatible.

Which is harder to learn: JQuery or JavaScript?

Depends on what you start with. It may be easier to learn JavaScript first because of how JQuery uses JavaScript functions. Any programming language will be difficult for first time coders but once you spend enough time with one the rest seem easy to pick up. It's recommended to learn a language to the fullest extent before working on another (if they serve the same purpose). For example, you might want to consider learning either JavaScript or TypeScript. It would be pointless to learn both at the same time.
